Image

Responsive image

In order to make your image responsive add .responsive-img class on the image tag.

Circular image

In order to make your image circular enclose it in a div tag of equal width and height and add .rounded-img class on the image tag.

Input

Input box have different states, some of these states can be success state where the user has added the correct input value, similarly there is unsuccessful state.

To use the above input style add .textbox class.

For success input style add .textbox .textbox-success class.

For unsuccessful input style add .textbox .textbox-fail class.

Snackbar

Snackbar can be used as popup messages.Snackbar can be primary,secondary and warning and error.

To use primary snackbar apply .snackbar .snackbar-primary class.

To use secondary snackbar apply .snackbar .snackbar-secondary class.

To use warning snackbar apply .snackbar .snackbar-accent class.

To use error snackbar apply .snackbar .snackbar-error class.

Grid

Grid can be used for basic alignment of content. To align content in two equal columns apply .grid-container .grid-2 class

To align content in three equal columns apply .grid-container .grid-3 class

To align content in sidebar and a main section apply .grid-container .grid-2-sidebar class

To align content in two sidebar and a main section apply .grid-container .grid-3-sidebar class

List

List can be of different type. To use a simple list i.e without any bullets or ordering use .list-style-none class on the ol or ul tag.
To use horizontal list apply .list-horizontal class.To use different style of list you can apply .list-style-circle, .list-style-square, .list-style-decimal, .list-style-zero-decimal
