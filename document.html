Alert

Alert dialog is used to alert the user with some different information. Alert can be of different types viz. success, warning, failure etc.

In order to use alert use the .alert class. For different styling use the following classes alongwith the .alert :

  • .alert-default for displaying default alert.
  • .alert-success for displaying successful alert.
  • .alert-failure for displaying unsuccessful alert.
  • .alert-warning for displaying warning alert.

Outlined Alert

Alert with just outlines can be used. To use outlined alert of different styles use the following classes alongwith .alert class.

  • .alert-default-outline for displaying default alert.
  • .alert-success-outline for displaying successful alert.
  • .alert-failure-outline for displaying unsuccessful alert.
  • .alert-warning-outline for displaying warning alert.

Button

Buttons are one of the main component of any application. Different style of button signifies the level of importance. To start using the button use the .btn class on the button element.

For styling the button component alongwith .btn class use the .btn-primary, .btn-secondary, .btn-accented, .btn-black, .btn-white classes. 

<button class="btn btn-primary ">Primary</button>
<button class="btn btn-secondary ">Secondary</button>
<button class="btn btn-accented ">Call to action</button>
<button class="btn btn-black ">Black button</button>
<button class="btn btn-white ">White button</button>

Grid

<div class="grid-container grid-2 primary-color">
  <div class="grid-item"></div>
  <div class="grid-item"></div>
</div>

To align content in three equal columns apply .grid-container .grid-3 class 


<div class="grid-temp-container grid-container grid-3 primary-color">

Input

Input box are used alot from searching to signing up forms. Input has different states, this states are great way to give feedback to the user.Some of these states can be success state where the user has added the correct input value,a failure state where the user has added wrong input value and a disabled state.

To use the above input style add .textbox class. This will give the default style when the input is in neutral state. And for the label give the .label class.

For success input style add .textbox .textbox-success class. And for the label use .label .label-success class. This can be used to show a valid input.

For unsuccessful input style add .textbox .textbox-fail class.For the label use .label .label-failure class. This can be used to show invalid input.

For disabled input style add .textbox .textbox-disabled class.For the label use .label .label-disabled class. This can be used to show disabled input.

For textarea input style add .textbox .textbox-textarea class. 

<label for="textbox-default" class="label">Default Label</label>
<input id="textbox-default" type="text " class="textbox margin-b-1r" placeholder="Something"/>

<label class="label label-success" for="textbox-success">Success Label</label>
<input id="textbox-success" type="text" class="textbox textbox-success margin-b-1r" placeholder="Success"/>

<label class="label label-failure" for="textbox-fail">fail Label</label>
<input id="textbox-fail" type="text" class="textbox textbox-fail margin-b-1r" placeholder="fail"/>

<label class="label label-disabled" for="textbox-disabled">disabled Label</label>
<input id="textbox-disabled" type="text" class="textbox textbox-disabled margin-b-1r" placeholder="disabled"/>

<label class="label" for="textbox-textarea">Textarea</label>
<textarea id="textbox-textarea" cols="30" rows="5" class="textbox textbox-textarea"></textarea>

Snackbar

Snackbar can be used as popup messages. Snackbar can be primary, secondary, warning and error. To start using snackbar apply .snackbar class. To style snackbar add the following classes :

To use primary snackbar apply .snackbar .snackbar-primary class.

To use secondary snackbar apply .snackbar .snackbar-secondary class.

To use warning snackbar apply .snackbar .snackbar-accent class.

To use error snackbar apply .snackbar .snackbar-error class.
